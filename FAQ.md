# Frequently Asked Questions

## General Questions

### What is Villages Reborn?
Villages Reborn is a Minecraft mod that enhances villages with cultural diversity, advanced AI-driven villager behaviors, dynamic events, and a rich crafting system.

### Which Minecraft versions are supported?
Currently, Villages Reborn supports Minecraft 1.21.4 with Fabric loader.

### Will there be a Forge version?
There are currently no plans for a Forge version as the mod heavily relies on Fabric-specific features.

## Features

### What cultures are available?
Currently implemented cultures include:
- Roman
- Egyptian
- Victorian
- NYC-style

### How do villagers use AI?
Villagers use Large Language Models to:
- Generate contextual dialogue
- Make decisions based on relationships
- Plan and participate in events
- Remember past interactions
- Express cultural-specific behaviors

### What types of events occur?
Events include:
- Cultural festivals and celebrations
- Seasonal gatherings
- Trading fairs
- Community projects
- Special ceremonies based on village culture

## Performance

### Will this mod impact my game performance?
The mod includes automatic system analysis and optimization. Impact varies based on:
- Your hardware specifications
- Number of active villages
- AI model settings
- Selected features

### How much RAM is recommended?
- Minimum: 4GB allocated to Minecraft
- Recommended: 8GB for optimal performance
- More may be needed for multiple active villages

## Gameplay

### How do I start interacting with villagers?
- Approach any villager in a cultural village
- Right-click to initiate conversation
- Build relationships through regular interactions
- Participate in village events

### How does the crafting system work?
- Unlock recipes by building relationships
- Use cultural crafting stations
- Collect culture-specific materials
- Complete village events for special recipes

### Can I customize village generation?
Yes, through the mod settings you can:
- Adjust village frequency
- Select preferred cultures
- Modify village size
- Configure structure generation

## Common Issues

### Villages aren't generating with cultures
- Check if structures are enabled in world settings
- Verify mod installation
- Ensure no conflicts with other village mods
- Try creating a new world

### Villagers aren't responding
- Check AI configuration
- Verify internet connection
- Ensure API keys are valid
- Try reloading the world

### Events aren't triggering
- Check event frequency settings
- Verify server tick rate
- Ensure proper world time
- Check for schedule conflicts
