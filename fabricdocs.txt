TITLE: Defining Sheep Shear Callback Interface (Java)
DESCRIPTION: Defines the `SheepShearCallback` interface, which event listeners must implement. It includes the static `EVENT` field, an array-backed event instance, and the `interact` method signature that listeners will provide. The invoker logic iterates through listeners and handles `ActionResult` return values.
SOURCE: https://github.com/fabricmc/fabric-docs/blob/main/versions/1.21/develop/events.md#_snippet_3

LANGUAGE: java
CODE:
```
package com.example.docs.event;

import net.fabricmc.fabric.api.event.Event;
import net.fabricmc.fabric.api.event.EventFactory;
import net.minecraft.entity.passive.SheepEntity;
import net.minecraft.entity.player.PlayerEntity;
import net.minecraft.util.ActionResult;
import net.minecraft.util.Hand;

/**
 * Callback for when a player attempts to shear a sheep.
 * Called before the sheep is sheared.
 *
 * {@link ActionResult#PASS} will continue with the shearing.
 * {@link ActionResult#FAIL} will cancel the shearing.
 * {@link ActionResult#SUCCESS} will cancel the shearing and signal a successful interaction.
 */
public interface SheepShearCallback {
    Event<SheepShearCallback> EVENT = EventFactory.createArrayBacked(SheepShearCallback.class,
            (listeners) -> (player, sheep, hand) -> {
                for (SheepShearCallback listener : listeners) {
                    ActionResult result = listener.interact(player, sheep, hand);

                    if (result != ActionResult.PASS) {
                        return result;
                    }
                }

                return ActionResult.PASS;
            });

    ActionResult interact(PlayerEntity player, SheepEntity sheep, Hand hand);
}
```

TITLE: Defining Fabric Event Callback Interface (Java)
DESCRIPTION: Defines the `SheepShearCallback` interface for a sheep shearing event. It includes an `Event` instance and the `interact` method that listeners must implement, using `ActionResult` to control event flow.
SOURCE: https://github.com/fabricmc/fabric-docs/blob/main/develop/events.md#_snippet_0

LANGUAGE: java
CODE:
```
package com.example.docs.event;

import net.fabricmc.fabric.api.event.Event;
import net.fabricmc.fabric.api.event.EventFactory;
import net.minecraft.entity.passive.SheepEntity;
import net.minecraft.entity.player.PlayerEntity;
import net.minecraft.util.ActionResult;
import net.minecraft.util.Hand;

/**
 * Callback for when a sheep is sheared.
 * Called before the wool is dropped.
 * Successful results will cancel the default wool drop.
 * PASS: Continue with the default wool drop.
 * SUCCESS: Cancel the default wool drop.
 * FAIL: Cancel the default wool drop.
 */
public interface SheepShearCallback {
	Event<SheepShearCallback> EVENT = EventFactory.createArrayBacked(SheepShearCallback.class,
		(listeners) -> (player, sheep, hand) -> {
			for (SheepShearCallback listener : listeners) {
				ActionResult result = listener.interact(player, sheep, hand);

				// If the result is not PASS, it means the event was handled.
				if (result != ActionResult.PASS) {
					return result;
				}
			}

			// If no listener handled the event, pass it on.
			return ActionResult.PASS;
		});

	ActionResult interact(PlayerEntity player, SheepEntity sheep, Hand hand);
}
```
