# Contributing to Villages Reborn

We welcome contributions to Villages Reborn! Here's how you can help:

## Reporting Bugs

If you find a bug, please report it on our [issue tracker](https://github.com/Beeny/villagesreborn-template-1.21.4/issues).

## Suggesting Features

If you have a suggestion for a new feature, please submit it on our [issue tracker](https://github.com/Beeny/villagesreborn-template-1.21.4/issues).

## Contributing Code

If you'd like to contribute code, please follow these steps:

1.  Fork the repository.
2.  Create a new branch for your feature or bug fix.
3.  Make your changes.
4.  Submit a pull request.

## Code Style

Please follow the code style used in the project.
