# Villager Cultural Texture Requirements

This document outlines the requirements for creating cultural variants of villager textures.

## General Requirements
- Base texture size: 64x64 pixels
- Format: PNG with transparency
- Must maintain vanilla villager proportions for compatibility
- Include profession markers (e.g., job badge areas)

## Cultural Variants

### Egyptian Villager (`egyptian.png`)
- Base color: Sandstone tones (#DED0B4)
- Clothing: Sandy tunics with draping
- Accessories: 
  - Scarab jewelry accents
  - Gold trim details
  - Optional headdress elements
- Profession markers should use Egyptian-themed icons

### Roman Villager (`roman.png`)
- Base color: White/cream (#F5F5F5)
- Clothing: 
  - Draped toga with folds
  - Optional Roman tunic underneath
- Accessories:
  - Gold laurel crown variations
  - Optional belt/sash
- Profession markers using Roman symbols

### Victorian Villager (`victorian.png`)
- Base color: Dark wool tones (#463E3F)
- Clothing:
  - Tailored vests with buttons
  - Long coats or jackets
- Accessories:
  - Top hats
  - Optional pocket watches
- Profession markers with Victorian industrial themes

### NYC Villager (`nyc.png`)
- Base color: Modern fabric tones (#4A4A4A)
- Clothing:
  - Hooded sweatshirts
  - Modern casual wear
- Accessories:
  - Sneakers/modern footwear
  - Urban-style accessories
- Profession markers with contemporary icons

## Profession Compatibility
Each texture must include areas for profession-specific markers:
- Badge area on the chest
- Optional head decoration area
- Tool/item holding positions

## File Naming
- egyptian.png
- roman.png
- victorian.png
- nyc.png

## UV Mapping
Maintain the vanilla villager UV layout for compatibility:
- Head: 0,0,8,8
- Body: 16,16,24,24
- Arms: 40,16,44,24
- Legs: 0,16,4,24

These textures should reflect their respective cultural aesthetics while maintaining compatibility with Minecraft's villager model and animations.
