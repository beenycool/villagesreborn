## Installation Guide
1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.4
2. Install it in your preferred launcher
3. Download Villages Reborn from [Modrinth](https://modrinth.com/mod/villages-reborn)
4. Place the downloaded JAR file into your Minecraft mods folder
5. Launch the game with Fabric loader

## Features
### Cultural Diversity
- **Roman Villages**: Experience ancient Roman architecture with forums, bathhouses, and classical styling
- **Egyptian Villages**: Discover pyramid-inspired structures and desert-themed communities
- **Victorian Villages**: Explore elegantly designed neighborhoods with period-appropriate aesthetics
- **NYC-Style Villages**: Modern urban communities with contemporary architecture
