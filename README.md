# Villages Reborn

Villages Reborn is a Minecraft mod that breathes new life into your villages! Explore culturally rich and diverse communities with villagers that have unique personalities and relationships.

## Key Features:

- **Explore Diverse Cultures**: Discover villages inspired by Roman, Egyptian, Victorian, and NYC themes. Each village boasts unique architecture, decorations, and cultural elements reflecting their heritage.
- **Meet Intelligent Villagers**: Interact with villagers powered by advanced AI. They have distinct personalities, form relationships, remember past interactions, and engage in culturally appropriate activities.
- **Experience Dynamic Events**: Immerse yourself in village-specific events and festivals that reflect the unique culture of each community, from Roman forums to Egyptian ceremonies.
- **Master the Cultural Crafting System**: Unlock special crafting recipes through your relationships with villagers, creating unique cultural artifacts and items specific to each village type.
- **Witness Advanced Social Dynamics**: Observe how villagers form bonds, develop emotions, and create a thriving social ecosystem with complex relationships and community roles.
- **AI-Powered Villager Behavior**: The mod leverages Large Language Models (LLMs) to drive advanced villager behaviors, bringing unprecedented realism to your Minecraft world.
- **Optimized Performance**: The mod analyzes your system specifications to optimize AI performance, ensuring a smooth and enjoyable experience regardless of your hardware capabilities.

## Install

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.4
2. Install it in your favorite launcher (such as the official Minecraft Launcher, MultiMC, or Prism Launcher)
3. Download the latest Villages Reborn release from [Modrinth](https://modrinth.com/mod/villages-reborn) 
4. Place the downloaded JAR file into your Minecraft mods folder:
   - Windows: `%appdata%\.minecraft\mods`
   - macOS: `~/Library/Application Support/minecraft/mods`
   - Linux: `~/.minecraft/mods`
5. Launch the game with Fabric loader
6. On first launch, you'll be prompted to select your preferred setup mode

## Configuration

Villages Reborn offers several configuration options:

- **LLM Settings**: Configure AI integration for advanced villager behaviors
  - API key configuration (if using external LLMs)
  - Model selection and parameters
  - System performance adjustments
  
- **General Settings**: Toggle features and customize the mod experience
  - Village generation frequency and size
  - Event occurrence rate
  - Villager interaction complexity
  - UI customization options

Access settings through the in-game menu or by editing configuration files in the config folder:
- `villagesreborn_general.json`: General mod settings
- `villagesreborn_llm.json`: AI configuration
- `villagesreborn_ui.json`: User interface options
